In addition, we use the forward mode notation in asin_forward for @[@ \begin{array}{rcl} Q(t) & = & 1 \mp X(t) * X(t) \\ B(t) & = & \sqrt{ Q(t) } \end{array} @]@ We use @(@ q @)@ and @(@ b @)@ for the p-th order Taylor coefficient row vectors corresponding to these functions and replace @(@ z^{(j)} @)@ by @[@ ( z^{(j)} , b^{(j)} ) @]@ in the definition for @(@ G @)@ and @(@ H @)@. \sum_{k=1}^{j-1} k b^{(k)} b^{(j-k)} \right) \\ z^{(j)} & = & \frac{1}{j} \frac{1}{ b^{(0)} } \left( j x^{(j)} - \sum_{k=1}^{j-1} k z^{(k)} b^{(j-k)} \right) \end{array} @]@ If @(@ j = 0 @)@, we note that @(@ F^{(1)} ( x^{(0)} ) = 1 / b^{(0)} @)@ and hence @[@ \begin{array}{rcl} \D{H}{ x^{(j)} } & = & \D{G}{ x^{(j)} } + \D{G}{ z^{(j)} } \D{ z^{(j)} }{ x^{(0)} } + \D{G}{ b^{(j)} } \D{ b^{(j)} }{ q^{(0)} } \D{ q^{(0)} }{ x^{(0)} } \\ & = & \D{G}{ x^{(j)} } + \D{G}{ z^{(j)} } \frac{1}{ b^{(0)} } \mp \D{G}{ b^{(j)} } \frac{ x^{(0)} }{ b^{(0)} } \end{array} @]@ If @(@ j > 0 @)@, then for @(@ k = 1, \ldots , j-1 @)@ @[@ \begin{array}{rcl} \D{H}{ b^{(0)} } & = & \D{G}{ b^{(0)} } + \D{G}{ z^{(j)} } \D{ z^{(j)} }{ b^{(0)} } + \D{G}{ b^{(j)} } \D{ b^{(j)} }{ b^{(0)} } \\ & = & \D{G}{ b^{(0)} } -
